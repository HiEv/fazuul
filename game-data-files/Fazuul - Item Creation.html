Combining objects is an extremely important method of discovery in Fazuul. Littered throughout the land, you will find these items frequently lying on the ground: gwingus, mongoo, snuge, torkus and wigglesnort. These 5 objects form the building blocks (level1-objects) to a number of new items in the game.

Because it would take a while to scour the land and find an object each time you need one, there are some objects which will generate each of these level1-objects for you, with the push of a button. By building all of these creation-objects, you will be able to construct more advanced objects. These important objects are:

Item Needed Item Created
Wufflar Torkus
Dampish Mongoo
Cistle Snuge
Suvar Wigglesnort
Nabob Gwingus


Helpful hint: Certain telnet clients will allow you to build macros, often with the Function keys on your keyboard. You can drop the creation-objects in a room, and build macros that will perform the following:
get the creation-object,
push the button [new object made],
drop the creation-object,
pick up the new object
With a single push of F1, I can have a gwingus created & in my hands... F2 creates a mongoo, etc.. This will make the process of creating more advanced items a little faster & easier.
Additional Note: There is a limit to the number of items in a room and in your hands. I recommend finding an area of 5 rooms in a line where you can drop each object. So for instance, drop a wufflar, move east, drop a dampish, etc. Then just move to the proper room, hit your macro and collect your items. This will avoid the item limit.

Also, take a look at my Fazuul Telnet Client which will automate the item creation process for you!

Next, here are the charts that show what objects will combine with other objects. Combinations are not order specific, so attaching a mongoo to gwingus is the same as attaching a gwingus to a mongoo. Also, in my experience, it seemed that the combinations worked in levels.. So level1-objects can only combine with other level1-objects.. level2 to level2, etc.
